# iOS Label Demo

## 简介

展示 UILabel 的各种用法：字体、颜色、对齐方式、多行文本。

## 启动和使用

在 Xcode 中打开 `ios-label-demo.xcodeproj`，运行即可。

## 教程

### UILabel 属性

```swift
label.text = "文本内容"
label.font = .systemFont(ofSize: 16)
label.textColor = .systemBlue
label.textAlignment = .center
label.numberOfLines = 0  // 多行
```
